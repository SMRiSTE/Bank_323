# Практическая работа №6. Часть 1
## Тестирование программных модулей методом «белого ящика»

### 1. Цель работы
* Провести тестирование разработанных программных модулей с использованием средств автоматизации Microsoft Visual Studio методом "белого ящика".
* Изучить и применить встроенные средства документирования кода C# — XML-комментарии.

---

### 2. Описание тестируемого модуля
Основным объектом тестирования является класс `BankAccount`. Он содержит:
* **Поля**: `m_customerName` (имя клиента) и `m_balance` (текущий баланс).
* **Методы**: 
    * `Debit`: вызывается при снятии средств со счета.
    * `Credit`: вызывается при поступлении средств на счет.
* **Константы ошибок**: Специальные сообщения для исключений, используемые для точной идентификации причин сбоя.

---

### 3. Документирование кода
В проекте реализовано документирование с использованием специальных тегов XML, которые содержатся в комментариях и описывают свойства или методы:
* `<summary>` — общее описание.
* `<param>` — описание параметров метода.
* `<returns>` — описание того, что метод возвращает.

### 4. Реализация Unit-тестов
Тестирование реализовано в проекте `BankTests`. Минимальные требования включают атрибут `[TestClass]` для класса и `[TestMethod]` для каждого метода теста.

#### Тесты метода Debit:
1. **Debit_WithValidAmount_UpdatesBalance**: проверка списания допустимой суммы.
2. **Debit_WhenAmountIsLessThanZero_ShouldThrowArgumentOutOfRange**: проверка реакции на отрицательную сумму.
3. **Debit_WhenAmountIsMoreThanBalance_ShouldThrowArgumentOutOfRange**: проверка реакции на сумму, превышающую баланс.

#### Тесты метода Credit (Самостоятельная работа):
1. **Credit_WithValidAmount_UpdatesBalance**: проверка зачисления валидной суммы.
2. **Credit_WhenAmountIsLessThanZero_ShouldThrowArgumentOutOfRange**: проверка выброса исключения при отрицательном зачислении.

### 5. Результаты выполнения
В ходе тестирования была обнаружена ошибка: сумма списания прибавлялась к балансу счета вместо того, чтобы вычитаться. Ошибка была исправлена в исходном коде класса `BankAccount`.
